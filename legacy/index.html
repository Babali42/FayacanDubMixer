Intro :

Dub Mix by Fayacan Son Corp. est probablement l'un des produits les plus novateurs dans cette galaxie que l'on appelle l'Internet.

Conçu à des fins de fun illimité, cet appareil virtuel vous permettra de vous identifier à vos producteurs de Reggae and Dub Music favoris tel que Fayacan Son original sound.

Le son :

L'instrumentale est composé en full digital style sur le logiciel FL Studio. La musique, d'une structure volontairement simple, permet à l'utilisateur de comprendre au plus vite les mécaniques du mixer.

Ces mécaniques sont les suivantes:
-Gestion du volume de chaque piste
-Activation d'un effet par piste

Cependant, pour passer un bon moment entre amis devant Dub Mix by Fayacan Son Corp. il était nécessaire d'y intégrer des effets "porcins" ou encore "éclatés": réverbération sur les snares, cut off sur les basses, doublage du kick. Chacun aura remarqué ce superbe phaser sur la piste "Melodic".

Le saviez-vous? Un export en MP3 de faible qualité vous assure des temps de chargements balayant toute concurrence!

L'application web :

Cette application web a été réalisée avec les technologies classiques du web : html css et js.

Afin de ne pas réinventer la roue nous avons utilisé des bibliothèques javascript comme Three.js pour la 3D, Pizzicato.js pour le son et Intro.js pour le tutoriel.

On a ensuite rassemblé le modèle 3D, conçu dans Blender, avec les pistes sons de Fayacan pour créer la table de mixage.

Le plus compliqué c'était les translations des faders et rotations des potards, on a dû faire des maths !

Enfin le petit écran du mixeur, et l'animation badante (Hexagone by BigWIngs) en fond d'écran, sur PC, sont des shaders webgl. Parfait pour bien kiffer !

Pour finir, si tu veux en savoir plus, le code source est disponible sur github.
